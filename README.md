### Installation :

```bash
pkg update && pkg upgrade -y
pkg install git -y
pkg install python -y
git clone https://github.com/ShTasrif/SH-CUSTOM-SMS
pip install requests
cd SH-CUSTOM-SMS
python csms.py
```
### Run Tools

```bash
csms
```
